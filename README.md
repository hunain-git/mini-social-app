MINI SOCIAL M

A simple social media web application built using HTML, CSS, and JavaScript. Users can sign up, log in, create posts, like/dislike posts, edit/delete posts, and toggle light/dark theme.

FEATURES: 
1) User Authentication (Sign Up / Login)
2) Create text and image posts
3) Like / dislike posts
4) Edit and delete posts
5) Search and sort posts
6) Profile picture support
7) Light/Dark mode toggle
8) Responsive design
